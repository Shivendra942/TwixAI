# TwixAI
TwixAI is a Django-based microblogging web app with secure user authentication and full tweet CRUD functionality. It features an AI-powered chatbot using the Google Gemini API to provide instant user assistance through real-time interaction, demonstrating seamless integration of modern AI with a scalable web application.
